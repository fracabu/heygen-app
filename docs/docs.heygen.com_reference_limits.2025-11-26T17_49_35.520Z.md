## What is an API Credit?   [Skip link to What is an API Credit?](https://docs.heygen.com/reference/limits\#what-is-an-api-credit)

API Credits are used when performing various tasks like creating videos, translating content, or using [Interactive Avatars](https://www.heygen.com/interactive-avatar). The credit usage depends on the endpoint:

| **Functionality** | **Credit Cost** | **Details** |
| --- | --- | --- |
| `video/generate` (Photo Avatar) | 1 credit / min | Charged in 30 sec increments |
| `video/generate` (Video Avatar) | 2 credits / min | Charged in 30 sec increments |
| `video_translate` | 3 credits / min (`fast`mode)<br>6 credits / min (`quality`mode) | Charged in 30 sec increments |
| `generate-proofread` | 1 credit / 2 min | 2-min video = 1 credit |
| `/v2/video/av4/generate` | 6 credits / min |  |
| Active streaming session | 0.2 credits / min | 30 sec min, 1 sec increments after |
| `photo_avatar/photo/generate` | 1 credit per call |  |
| `photo_avatar/look/generate` | 1 credit per call |  |
| `photo_avatar/train` | 4 credits per call |  |
| `photo_avatar/add_motion` | 1 credit per call |  |
| `photo_avatar/add_sound_effect` | Free |  |
| `photo_avatar/upscale` | 0.5 credits per call |  |

> _Check your usage histoy at: [Space Settings â†’ History â†’ API Usage](https://app.heygen.com/settings?from=&nav=History)_

## Photo Avatar and Look Generation Limits   [Skip link to Photo Avatar and Look Generation Limits](https://docs.heygen.com/reference/limits\#photo-avatar-and-look-generation-limits)

### Photo Generation (API vs Web)   [Skip link to Photo Generation (API vs Web)](https://docs.heygen.com/reference/limits\#photo-generation-api-vs-web)

- You can generate **unlimited photos** using the [API](https://docs.heygen.com/reference/generate-photo-avatar-photos).
- However, the **HeyGen Web App** has a **limit of 400 photo generations per month**.
- If you **exceed 400 total photos** (including those created via API), the **Web App will block** further photo generation until the next month.
- You can **still continue generating photos via API**, even after reaching the web limit.

### Avatar Looks per Group   [Skip link to Avatar Looks per Group](https://docs.heygen.com/reference/limits\#avatar-looks-per-group)

- Max **500** Looks per Avatar Group.

## `video/generate` Endpoint Limits   [Skip link to [object Object], Endpoint Limits](https://docs.heygen.com/reference/limits\#videogenerate-endpoint-limits)

### Resources   [Skip link to Resources](https://docs.heygen.com/reference/limits\#resources)

HeyGen limits the size and format of resources such as images, audio or video, that can be uploaded to the Video/Generate endpoint. Render errors may occur if resources are invalid.

| Resource Type | Supported format | Max File Size | Resolution |
| --- | --- | --- | --- |
| Video | MP4 | 100MB | <2K |
| Image | JPG, PNG | 50MB | <2K |
| Audio | WAV, MP3 | 50MB |  |

Please ensure:

- Your resource _URL is public and accessible to anyone_.
- The provided _asset is in one of the supported formats_.
- Check that the _asset file extension aligns with the actual format_.
- Verify that the _asset file is not corrupted or misformatted_.

### Video Avatar Input   [Skip link to Video Avatar Input](https://docs.heygen.com/reference/limits\#video-avatar-input)

Avatar input can be either text or audio:

- The [Input Text](https://docs.heygen.com/reference/create-an-avatar-video-v2#textvoicesettings) \- which is the text the Avatar speaks - should be less than 5,000 characters.
- The duration of [audio input](https://docs.heygen.com/reference/create-an-avatar-video-v2#audiovoicesettings) for avatars should not exceed 10 minutes (3600 seconds).

Invalid inputs will result in video render failures.

### Output Video Specifications   [Skip link to Output Video Specifications](https://docs.heygen.com/reference/limits\#output-video-specifications)

- The frame rate of video output containing avatars is 25fps.
- `dimension` parameter width or height must be less than `4096` and greater than `128`.
- Maximum 50 [Video Input](https://docs.heygen.com/reference/create-an-avatar-video-v2#videoinput)/scenes allowed per video.
- WebM Format: Available only to **paid API plans**.
