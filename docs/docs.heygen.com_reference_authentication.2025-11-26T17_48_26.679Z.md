# Authentication

Ask AI

Authenticate with HeyGen API

To access [HeyGen's API](https://www.heygen.com/api-pricing) and perform operations, you must authenticate and authorize your requests using the HTTP `X-Api-Key` header.

To find your API Token, navigate to your account's _Space_ by clicking the top left corner of the **[HeyGen App](https://app.heygen.com/)** home screen. From the dropdown menu, select [Space Settings](https://app.heygen.com/settings). Your API token will be displayed under the [API tab](https://app.heygen.com/settings?nav=API). This token will be used to authenticate your API requests.
