# Create Avatar IV Videos

Produce dynamic avatar videos via Create Avatar IV Video API

Create personalized avatar videos using the HeyGen Avatarâ€¯IV API. This guide walks you through generating avatar videos from a single photo with AI-powered motion, expressions, and voice.

# 1\. Upload Photo   [Skip link to 1. Upload Photo](https://docs.heygen.com/docs/create-avatar-iv-videos\#1-upload-photo)

Before generating a video, upload your photo using the [Upload Asset](https://docs.heygen.com/reference/upload-asset) API. This will return an `image_key`, which is required for the Avatarâ€¯IV video.

# 2\. Prepare Script and Voice   [Skip link to 2. Prepare Script and Voice](https://docs.heygen.com/docs/create-avatar-iv-videos\#2-prepare-script-and-voice)

Your video requires:

- Script - text that the avatar will speak.
- Voice - you must provide a `voice_id`, obtained from the [List All Voices (V2)](https://docs.heygen.com/reference/list-voices-v2) API.

Additionally, you can provide a custom audio recording using:

- `audio_url` \- URL of a custom audio file.
- `audio_asset_id` \- HeyGen asset ID of an audio file.

## Customize Motion   [Skip link to Customize Motion](https://docs.heygen.com/docs/create-avatar-iv-videos\#customize-motion)

Avatarâ€¯IV Video also supports custom motion prompts, allowing you to guide the avatarâ€™s gestures and expressions. Use descriptive prompts in the `custom_motion_prompt` field. Set `enhance_custom_motion_prompt` to **true** to let AI refine the motion.

Motion results depend on the script and prompt; the API does not require predefined gestures.

# 3\. Generate Video   [Skip link to 3. Generate Video](https://docs.heygen.com/docs/create-avatar-iv-videos\#3-generate-video)

Make a POST request to the [Create Avatar IV Video](https://docs.heygen.com/reference/create-avatar-iv-video) API with the required parameters: `image_key`, `video_title`, `script`, and `voice_id`.

For full request and response details, see detailed API reference: [Create Avatar IV Video](https://docs.heygen.com/reference/create-avatar-iv-video).
