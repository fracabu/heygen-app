# Quick Start

Get started with HeyGen API.

Welcome to HeyGen! ðŸ‘‹

This guide shows you how to build videos programmatically with AI avatars and voices, translate them into other languages, or even power real-time streaming avatars for interactive use cases, and more.

# Getting Started with HeyGen APIs   [Skip link to Getting Started with HeyGen APIs](https://docs.heygen.com/docs/quick-start\#getting-started-with-heygen-apis)

This Quick Start guide is designed to give you just enough orientation to start experimenting. It covers:

- How to configure HeyGen authentication
- How to use the HeyGen Postman collection
- Links to advanced guides and recipes.

> ðŸ“˜
>
> ### Note:   [Skip link to Note:](https://docs.heygen.com/docs/quick-start\#note)
>
> The **HeyGen API** is available as a standalone service, separate from the **HeyGen App**.
>
> - Your **HeyGen App plan** \- **Free**, **Creator**, **Team**, or **Enterprise** \- controls what you can access in the [HeyGen web application](https://app.heygen.com/).
> - All **HeyGen App users** automatically receive **Free Trial API access**, which includes Video Generation, Video Translation, and Interactive Avatar endpoints.
> - For higher usage, the API offers three paid subscription tiers: **Pro**, **Scale**, and **Enterprise**.
>
> _For more information on pricing and usage, please refer to our [limits page](https://docs.heygen.com/reference/limits#basic-api-vs-advanced-api)._

# Step 1: Obtain your API Key   [Skip link to Step 1: Obtain your API Key](https://docs.heygen.com/docs/quick-start\#step-1-obtain-your-api-key)

1. Register or sign in to your [HeyGen account](https://app.heygen.com/login). This is the first step to make professional avatar videos by leveraging HeyGen's powerful AI capabilities.

2. Go to [Settings > Subscriptions & API > HeyGen API > API token](https://app.heygen.com/settings?from=&nav=Subscriptions%20%26%20API).
![](https://files.readme.io/823b964538f3fd697dfe7e0707d16958a903804a44d12c75f665ae0b103ddc45-sub-modified.png)

You will use this key in all API requests.

> â—ï¸
>
> **Caution**
>
> Keep your API token secure. Do not expose it in client-side code. Store it in environment variables or a secrets manager.

Every request to the HeyGen API must be authenticated with an API key. You need to include the key in the request header as X-API-KEY.

> ðŸ“˜
>
> **Note:**
>
> For more information on:
>
> - Use of API key, see the [Authentication](https://docs.heygen.com/reference/authentication) page.
> - Usage quotas, request limits, and subscription tiers, see the [Limits](https://docs.heygen.com/reference/limits) page.

# Step 2: Try the API in docs, Postman, or your client   [Skip link to Step 2: Try the API in docs, Postman, or your client](https://docs.heygen.com/docs/quick-start\#step-2-try-the-api-in-docs-postman-or-your-client)

You can run HeyGen API requests in several ways:

- **Directly in the API docs:** Each endpoint in the [HeyGen API Reference guide](https://docs.heygen.com/reference/)
offers an interactive console to the right, where you can enter your API key and try requests immediately.
- **Postman Collection:** Import the official [HeyGen Postman Collection](https://docs.heygen.com/reference/postman-collections)
to explore request templates you can fill in with your own values and run instantly.
- **Your own terminal or client:** Send requests from your terminal or any HTTP client.
